# 📖 О проекте

**Squid Game Shooter** — игра, разработанная на **Unity** с использованием **C#** в рамках курса **«Основы программирования на C#»**.

Игрок оказывается в мире, вдохновленном сериалом **«Игра в кальмара»**, где управляет заключенным в зеленой форме. Главная задача — собрать как можно больше денег и выбраться живым, преодолевая сопротивление вооруженного персонала.

Проект создавался с упором на:

- 🎯 интересный игровой процесс;
- 🎨 визуальный стиль и атмосферу;
- 🤖 поведение противников;
- 🔫 разнообразие оружия и тактик прохождения.

---

# 🎮 Геймплей

Игрок исследует комнаты комплекса в поисках денег.

В каждой комнате находятся:

- 💰 сундуки с деньгами;
- 📦 ящики с новым оружием;
- 👮 вооруженный персонал, охраняющий добычу.

Особенности игры:

- уничтожайте противников;
- разрушайте ящики и находите более мощное оружие;
- самостоятельно выбирайте порядок исследования комнат;
- подстраивайтесь под особенности поведения разных типов врагов;
- используйте окружение и оружие для выживания.

## 🎯 Цель игры

> Собрать максимальное количество денег и остаться в живых.

# 🏗 Архитектура проекта

## 🤖 AI противников

Персонал управляется собственной системой **BotAI**, которая отвечает за:

- обнаружение игрока;
- преследование;
- атаку;
- смену состояний поведения.

---

## 📦 Лут-система

После разрушения сундуков и ящиков могут выпадать:

- 💰 деньги;
- 🔫 оружие;
- ⭐ улучшения.

Это делает каждое прохождение немного уникальным.

---

## 🎨 Анимация

Для персонажей использована **костная анимация Unity**, благодаря которой реализованы:

- бег;
- атаки;
- смерть;
- анимации противников.
